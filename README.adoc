# Service Discovery Demo for Prometheus on okd

## Prerequisites

Default Minishift installation.

## Setup

```bash
$ oc apply -f s2i-java11.yml
$ oc apply -f demo.yml
$ oc apply -f prometheus
$ oc start-build demo --from-dir=demo
```
